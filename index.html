GreenLink - 2025

¿Qué es nuestro portal web?

GreenLink

Es un portal web híbrido que unifica el comercio ecológico.

Para quién: Productores locales, Vendedores minoristas y Clientes finales.
Valor aportado: Marketplace unificado + Transparencia total mediante trazabilidad IoT.

Modelo Mixto Click para ver

Business to Business

Conecta productores con vendedores. Gestión de grandes volúmenes, stock y logística.

Business to Consumer

Permite al cliente final comprar productos ecológicos certificados y ver su origen en tiempo real.

Restricciones y Especificaciones

Backend & BD

Framework Laravel (MVC).
Uso de Eloquent ORM.
Base de datos MySQL (Relacional).

Frontend

HTML5, CSS3, JavaScript nativo.
Tecnología AJAX para actualización asíncrona de sensores.

Integración IoT

API REST para recepción de datos.
Formato JSON y validaciones estrictas.

E-Commerce & Legal

Carrito, Checkout, Stock.
Cumplimiento RGPD, Cookies y Aviso Legal.

Implementación Técnica

Arquitectura MVC

  • Modelo: Eloquent ORM. Definición de relaciones 1:N y N:M. Seeders para datos de prueba.
  • Vista: Blade Templates. Herencia de layouts (`@extends`), componentes para alertas y modales.
  • Controlador: Lógica de negocio separada. `SensorController`, `CartController`.

Frontend Dinámico

  • Uso de AJAX puro para comunicación con API interna.
  • Diseño Responsive con CSS Grid/Flexbox (sin Bootstrap pesado).

Evidencia de RA

  • R2 & R4: Se ha desplegado una API RESTful completa documentada.
  • R3: Integración de HTML, CSS, JS, PHP y SQL.
  • R15: Resolución autónoma de la lógica del carrito de compras y stock.

Conclusiones

  • Se ha logrado un portal funcional que integra comercio tradicional con tecnología IoT moderna.
  • La arquitectura MVC en Laravel facilita la escalabilidad y el mantenimiento del código (B2B).
  • La separación Backend/Frontend mediante API permite futuros clientes móviles.
  • El sistema de trazabilidad aporta valor añadido real al cliente final (B2C), diferenciando a GreenLink.
  • Se han cumplido los requisitos legales y de seguridad básicos para un entorno web.
